# Math Challange

## Calculate Exponentiation and the Root of X

O programa deverá receber 2 input INT

- X que será o valor
- Y o numero da potencia

Deverá ser capaz de executar somente Potencia ou Raiz, a depender da entidade.

### Regras

- Somente variaveis primitivas
- Não poderá solicitar apoio de calculos internos pré prontos como Math.pow()
